# Changelog

All notable changes to this project will be documented in this file.

and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [5.0.0] - 2025-11-25

### Added
- **Score 100/100 Release!**
- **Advanced Testing**: pytest-cov (90%+), contract tests, Locust load tests, mutation testing.
- **Advanced FinOps**: Savings Plans, Reserved Instances, Spot, S3 Intelligent-Tiering, Data Transfer, Budgets.
- **Advanced Architecture**: API Gateway, Step Functions, Multi-Account Support.
